⚙️ Cloud Sync — Supabase

Connect to Supabase so your data syncs between your iPhone and laptop automatically.

How to get these details:
1. Go to supabase.com → Sign up free
2. Create a new project (any name)
3. Go to Settings → API
4. Copy Project URL and anon public key
5. Paste them above and tap Connect

One-time database setup:
In Supabase → SQL Editor, run:
create table if not exists app_data ( user_id text primary key, data jsonb not null default '{}', updated_at timestamptz default now() );
alter table app_data enable row level security;
create policy "Allow all" on app_data for all using (true) with check (true);
